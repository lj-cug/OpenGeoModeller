# RegESM

区域性地球耦合系统
包括：RegCM/WRF + ROMS/MITgcm + WAM (Wave Model) + RTM (Hydrological River Model) + COP (Co-processing by ParaView)

ESMF耦合器+ ParaView Offscreening + In-situ visualization(原位可视化)

安装GTC-2018-demo的bash脚本程序
