#  ESM (Earth System Modelling)

区域性地球耦合模拟系统

## CESM

全球尺度的地球耦合模拟系统，在HPC领域具有广泛的开发社区

## RegESM

区域性地球耦合模拟系统，具备Catalyst的在线可视化功能，观察快速移动物理过程，如热带风暴

## COAWST

USGS开发的近海岸环境的区域性ESM系统

# Coupler

地球系统模拟耦合器，包括：ESMF, BMI, OASIS3-MCT, ...

耦合器的综述参考：
 
Bert Jagers. Linking Data, Models and Tools: An Overview. 5th International Congress on Environmental Modelling and Software - Ottawa, Ontario, Canada - July 2010
