# LoopStructural

澳大利亚开发的隐式地质建模工具，Python语言

基于SURFE三角网格化地质体表面

基于lavavu库作为可视化后端

## map2loop

配套的将数值化地质图转换为LoopStructural的输入文件的工具，但主要是针对西澳大利亚的公开地质数据库

用于我国的情况时，需要修改Python脚本，且目前仅有一个公开的地质数据库，且是MapGIS格式的，需要转换为ArcGIS的矢量格式文件：

李晨阳，王新春，何春珍，吴轩，孔昭煜，李晓蕾. 2019. 全国1∶200 000 数字地质图（公开版）空间数据库.中国地质，46(S1):1-10.

## 地震解释输入

基于地震数据解释得到的地震属性，作为LoopStructural的输入文件

## LoopStructural的参考文献

Lachlan Grose, Laurent Ailleres, Gautier Laurent, et al. LoopStructural 1.0: time-aware geological modelling. Geosci. Model Dev., 14, 3915C3937, 2021
