# tRIBS
2003年Vivoni的 MIT博士学位论文的基础上发展来的非结构网格的分布式水文模型.

近年来，得到了快速发展，已经开发，且使用Python语言的前处理工具pyTRIBS

C++语言编程

基于TIN网格, METIS区域分解的MPI并行化

## 参考文献
Enrique R. Vivoni. 2003. Hydrologic Modeling using Triangulated Irregular Networks: Terrain Representation, Flood Forecasting and Catchment Response. MASSACHUSETTS INSTITUTE OF TECHNOLOGY.
