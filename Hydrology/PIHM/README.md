# PIHM

基于求解常微分方程的sundials库，使用三角形非结构网格，实施分布式水文模拟降雨-径流等过程。

编程语言： C

OpenMP并行

数据同化

## 输入数据处理

分布式水文模型都需要大量不同类型的输入数据，处理过程繁琐

PIHM模型的快速的数据处理，可联系舒乐乐课题组(PIHM的开发课题组人员):

https://www.shud.xyz/zh/
