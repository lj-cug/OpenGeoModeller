# CHM

加拿大开发的冰雪下垫面的分布式水文模型，基于非结构网格，体现局部地形等差异

可以使用mesher作为前处理工具，生成考虑诸多下垫面因素比重的三角形网格

OpenMP多线程并行化

https://github.com/Chrismarsh/CHM

网格生成工具：https://github.com/Chrismarsh/mesher
