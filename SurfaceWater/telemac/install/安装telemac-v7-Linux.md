# make编译源码

## 下载源码

wget https://gitlab.pam-retd.fr/otm/telemac-mascaret/-/archive/v7p3r1/telemac-mascaret-v7p3r1.tar.gz

## 解压缩

tar xf telemac-mascaret-v7p3r1.tar.gz

cd telemac-mascaret-v7p3r1

## 设置环境变量

gedit setenv.sh

## 编译

make
