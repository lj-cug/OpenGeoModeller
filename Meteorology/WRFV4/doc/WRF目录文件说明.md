# WRF的路径文件说明

```
Makefile             编译用      
README               关于WRF/ARW的一些信息(版本信息，编译安装信息等等)
README_test_cases    理想化试验说明
README.NMM           关于WRF/NMM的一些信息(版本信息，编译安装信息等等)
README.rsl_output    针对 NMM的说明
Registry/            WRF Registry文件
arch/                编译配置选项文件目录
clean                清理创建可执行文件的脚本
compile              编译WRF源码的脚本
configure            创建的configure.wrf文件脚本
chem/                化学模块目录
dyn_em/              WRF动力核心
dyn_exp/             '玩具'动力核心
dyn_nmm/             NMM动力核心和算法
external/            扩展包目录(I/O相关, ESMF接口等)
frame/               WRF框架模块
inc/                 头文件
main/                主程序
phys/                物理模块目录
run/                 运行WRF的地方
share/               WRF中间层的WRF I/O模块
test/                测试算例
tools/               工具目录
```
