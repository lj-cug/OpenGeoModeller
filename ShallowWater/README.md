# ShallowWater

## 仓库内容
   
   浅水方程(Shallow Water Equation)的求解框架及模型，包括：
   
   1. Triton - CPU和GPU集群并行的结构网格下的洪水模拟
   2. Volna-OP2 - 基于特定域语言(DSL) OP2库的非结构网格海啸波传播模拟
   3. Titan2d - 基于MPI并行的滑坡模拟
   4. LISFLOOD-FP-8.0 - OpenMP与CUDA并行的间断Galerkin法的2D洪水模型
