## Titan2D

滑坡动力过程的MPI并行模拟
求解浅水方程，滑坡需要考虑源项的流变，主要模型有：
Mohr-Coulomb (MC)
Pouliquen-Forterre (PF)
Voellmy-Salm (VS)

### 特色

MPI集群并行
AMR技术
