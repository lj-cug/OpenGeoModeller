# OpenGeoModeller

#### Description
地球流体力学模拟系统，包含：气象、地表水（海洋、河流与湖泊）、地下水、地震波等组件。目的是：（1）快速编译安装和运行模式；（2）前处理、编译和后处理的文档说明；（3）高性能计算，提高时空分辨率；（4）自动化建模工作流。
