#  VirtualReality(VR)

## ParaView 和 Tecplot

CFD后处理常用的开源和商业的可视化软件

## In-situ-Visualization

在线可视化系统，包括：

1. ParaView-Catalyst
2. SENSEI
3. ADIOS
4. Ascent
5. VisIT

## AR-Sandbox

增强现实系统，用于地球科学的教学

### AR_Sandbox-FM

Delft3D Flexible Mesh 模型的AR演示系统，Python编程

### AR-Sandbox-UCDavies

UC Davies大学开发的浅水方程求解的AR演示系统，C++编程

### open-AR-Sandbox

德国人开发的地球科学AR演示系统，Python编程，包括：地震波传播、地质构造模型、地球物理等模块，功能最全

## High-Order-Viz

高阶数值格式计算结果，如DG法，的可视化

## Oculus-Quest

VR设备与ParaView的连接设置
