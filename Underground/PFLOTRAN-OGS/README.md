# PFLOTRAN
PFLOTRAN模型被用于评估化石能源资源深部地质构造所封存的二氧化碳对环境的影响，以及遗留废弃物场所的整治效果评
估等。PPFLOTRAN由美国洛斯阿拉莫斯(Los Alamos) 国家实验室支持开发，采用多级-多相-多成分地下反应流模型模拟地下污染物的流动

1. FORTRAN 2003面向对象编程
2. MPI并行化
3. PETSc-3.19库

## PFLOTRAN-OGS
PFLOTRAN-OGS是英国OpenGeosim公司研发的，开源的CO2存储和油气储层模拟器.
