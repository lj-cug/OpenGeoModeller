# LBDEMcoupling
耦合格子Boltzmann方法Palabos与离散元模型LIGGGHTS, 编程语言 C++  

可模拟床面泥沙颗粒的起动，以及水沙两相流的介观模型.

[palabos](http://www.palabos.org)

[仓库链接](https://github.com/philippeseil/LBDEMcoupling-public)

Seil (2017)论文给出了应用该模型模拟床面泥沙颗粒起动的算例.**仅有此1个算例**
