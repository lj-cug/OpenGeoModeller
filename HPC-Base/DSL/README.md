# DSL (Domain-Specific Language)

特定域语言, 一种将数值方法、并行化编程、应用开发等过程分离的技术, 让数学家、计算机程序员和专业人员独立工作, 并完美的结合在一起

## OP2

非结构网格的DSL库, 有2种方式：
``` 
1 source-to-source (Volna模型采用)
2 pyOP2 (已经融入firedrake框架)
```

## Devito

有限差分法的DSL库，在地球物理领域有很大影响力, 如基于Devito的FWI应用

以及使用Julia语言封装后的JUDI, 都是基于Devito核心库

## firedrake

有限单元法的DSL库, 在地球科学领域有很多应用, 例如： Thetis, sypro(医学成像的FWI应用)
