    线性方程组(Ax=b)求解器

	Nvidia公司开源的GPU集群并行求解器，与PETSc库结合使用：PETSc-Amgx

	俄国人开发的多重网格求解器，使用OpenCL实现异构并行
	
### paralution

	只能用于单节点计算，集群版本是收费的商业版本
