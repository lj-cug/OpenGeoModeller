# PETSc

PETSc是HPC领域常用的一个程序库，可在PETSc库基础上高效开发CFD代码

很多隐格式的CFD模型，涉及求解大规模的（稀疏系数矩阵的）线性方程组，可使用PETSc来管理数据结构和调用求解器，例如PETSc-AmgX
