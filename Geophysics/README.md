# Geophysics 

地球物理的模拟程序

## ASOFI3D

各向异性的地震波传播正演模型

## OpenSWPC

结构网格模式下的有限差分法的地震波传播正演模型

## defmod-swpc

地壳变形模拟defmod与地震波传播的有限差分模型OpenSWPC的耦合模拟

## Seissol

非结构网格模式的ADER-DG高阶格式的地震波传播正演模型

## Devito + JUDI

基于特定域语言Devito库，实施有限差分离散，以及Julia语言的JUDI库，实现全波形反演(FWI)全过程

## gprMAX

GPR(穿地雷达)的电磁波传播正演模型
