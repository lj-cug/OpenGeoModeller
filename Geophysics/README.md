# Geophysics 

地球物理的模拟程序

## OpenSWPC

结构网格模式下的有限差分法的地震波传播正演模型

## defmod-swpc

地壳变形模拟defmod与地震波传播的有限差分模型OpenSWPC的耦合模拟

## Seissol

非结构网格模式ADER-DG高阶格式的地震波传播正演模型

## Devito + JUDI

基于有限差分特定域语言Devito库，以及Julia语言开发的JUDI库，快速实现全波形反演(FWI)全过程

## spyro + firedrake

采用间断Galerkin法和波形自适应非结构网格技术的全波形反演模拟，特点是在非结构网格上求解波动方程
